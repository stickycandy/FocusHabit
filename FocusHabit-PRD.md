## 一、产品概述

### 1.1 产品名称
**FocusHabit**（专注习惯）

### 1.2 产品定位
一款将**习惯养成**与**专注计时**相结合的 iOS 应用，帮助用户通过番茄工作法培养良好习惯，提升个人效率。

### 1.3 核心价值
- 🎯 **双引擎驱动** - 习惯追踪 + 专注计时，相互促进
- 📊 **数据可视化** - 清晰看到成长和进步
- 🏆 **游戏化激励** - 成就系统保持动力
- 🎨 **极简设计** - 专注核心功能，无干扰

### 1.4 目标用户
| 用户类型 | 核心需求 |
|---------|---------|
| 学生党 | 专注学习、培养学习习惯 |
| 职场新人 | 时间管理、技能学习 |
| 自由职业者 | 工作效率、规律作息 |
| 健身爱好者 | 运动打卡、习惯坚持 |

## 二、功能设计

### 2.1 功能架构
```
FocusHabit
├── 习惯追踪（创建/打卡/统计）
├── 专注计时（番茄钟/白噪音/记录）
├── 数据分析（日历/图表/周报）
└── 成就系统（徽章/等级）
```

### 2.2 习惯追踪模块

**习惯创建：**
- 名称、图标（预设库）、颜色（6 主题色）
- 提醒时间（可设置多个）
- 重复频率（每天/每周特定几天）
- 关联专注模式（可选）

**打卡方式：**
1. 点击打卡按钮
2. 专注完成后自动打卡
3. 小组件快捷打卡

**反馈机制：**
- ✅ 成功动画（粒子效果）+ 震动反馈
- 🔥 连续天数提示
- 💬 随机鼓励语录

### 2.3 专注计时模块

**番茄钟设置：**
- 默认：25 分钟专注 + 5 分钟休息
- 自定义：15-60 分钟可调
- 长休息：每 4 轮后 15 分钟

**白噪音（6 种）：**
🌧️ 雨声 | 🌊 海浪 | 🔥 篝火 | ☕ 咖啡馆 | 🌲 森林 | 🎹 钢琴曲

**专注记录：**
- 时长、标签、评分（1-5 星）、关联习惯

### 2.4 成就系统

| 成就名称 | 解锁条件 |
|---------|---------|
| 🌱 新手上路 | 完成第 1 次打卡 |
| 🔥 持之以恒 | 连续打卡 7 天 |
| ⏱️ 专注达人 | 累计专注 100 小时 |
| 🏆 习惯大师 | 同时坚持 5 个习惯 |
| 💯 百日挑战 | 连续打卡 100 天 |

**等级系统：** Lv.1 新手 → Lv.6 传奇

## 三、技术架构

### 3.1 技术栈
| 模块 | 技术选型 |
|------|---------|
| 开发语言 | Swift 5.9+ |
| UI 框架 | SwiftUI |
| 数据持久化 | CoreData/SwiftData |
| 图表库 | Swift Charts |
| 数据同步 | iCloud CloudKit |

### 3.2 数据模型

**Habit（习惯）：** id, name, icon, color, reminderTimes, frequency, targetCount, createdAt

**FocusSession（专注）：** id, startTime, endTime, duration, status, relatedHabitId, rating

**HabitLog（打卡记录）：** id, habitId, completedAt, note
